A pet grooming website concept designed to showcase a soft, friendly, and modern brand experience for pet care services.

---

## About the Project

Fluff & Puff is a clean and playful website focused on pet grooming services.  
It highlights a warm and stress-free experience for pets while presenting services, packages, and testimonials in a modern layout.

This project was built to practice UI design, layout structuring, and frontend development using simple technologies.

---

## Features

- Clean and modern UI design
- Service and pricing sections
- Testimonials and gallery layout
- Responsive design for different screen sizes
- Soft, pet-friendly visual style
